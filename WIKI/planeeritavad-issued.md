# Planeeritavad issue'd

Kirjuta siia enne GitHub issue’de loomist oma peamised tööülesanded, küsimused või probleemid.

See fail aitab:
- mõelda töö enne läbi;
- jagada suure töö väiksemateks osadeks;
- otsustada, millised issue’d tuleb GitHubis päriselt avada.

## Kuidas seda faili kasutada?

- üks rida või plokk = üks võimalik issue;
- kirjuta lühidalt ja selgelt;
- kui issue on GitHubis loodud, võid lisada selle juurde lingi või märkuse.
